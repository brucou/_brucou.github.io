useful support tools for czech language study

Last modified on May 06, 2018 - czech language learning study tools 2 min read

Top language study tools

  • Memrise
  • czedlex : A Lexicon of Czech Discourse Connectives
  • Internetová jazyková příručka : put a word and get all its declined forms – useful to get the lemma, or check the conjugation of a word
  • dictionnary : czech-english and vice-versa
  • Treq : database of translation czech-english equivalent, pretty useful to understand usage, get synonims, and also take example from the corpus
  • Corpora : get examples of words in context as taken from rich corpora - also allows to analyze collocation! and access frequency information
  • Vallex 3.0 : analyze czech verbs valency – this gives all common usages for a verb!! byt for instance has 20 different patterns!!
    • morphological tagging reference : that helps makes sense of NNP4----AA--- stemming from morphological analysis of a word.
  • CzEngVallex : similar to Treq, except that more concise : just the possible translation ranked by frequency – more usable in the end, as less information to process
  • linguee : allows to enter an english idiom and see miscellaneous translations in czech. Pretty useful and simpler to use than corpus-based linguistically-minded tools from czech philology university.

Segundary language study tools

  • Verbalex : lexicon of verb valencies for the Czech language derived from Vallex 1.0, so preferable to use Vallex 3.0?
